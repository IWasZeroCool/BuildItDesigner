SimCity BuiltIt Designer

Version 1.0

SimCity BuildIt Designer is a tool to play with different configurations of city planning for the SimCity BuildIt game. Obviously, this has no affiliation with EA or the actual game. Statistics and logic within the planner are estimations and make no garuntee of accuracy.

Some elements are missing or may be inaccurate. These include some of the buildings and possibly some of the area effects. If you like the tool and wish to help, here's some things I could use help on:

  • Missing building data for transportation
  • Missing building data for entertainment
  • Missing building data for gambling
  • Missing building data for landmarks
  • Missing building data for worship
  • Verification of how area effects work when the size of the effect isn't divisible by the size of the structure
  • Beautification: The UI could use icons and images

If you would like to contribute to the project, you can fork the github repo here
